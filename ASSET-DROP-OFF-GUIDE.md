# Don't See Guide

Don't See is the shared Glasgow Branch hub for finding current work quickly and adding material when people only have small pockets of time.

Committee members can use the dashboard to open the area they are working on, then either:

- use the intake page to drag/drop files and add notes, or
- open the Assets Index to see, download or add assets.

The aim is simple: no more getting lost in different files, inboxes, chats or folders when someone needs to send an email, update a page, or check where a project has got to.

## How Updates Work

1. Open `RCOP-live-project-dashboard.html`.
2. Choose the relevant area.
3. Open the intake page.
4. Add notes, dates, links and files.
5. Use **Save to chosen folder**.
6. Select the matching asset folder if you are saving files locally.
7. Use `assets-index.html` to view, download or add collected assets from the browser.

## Downloading Assets

Open `assets-index.html`, find the relevant area, then click **Download** on the asset you need.

## Uploading Assets

Open `assets-index.html`, find the relevant area, drag files into the upload box, then click **Save to asset folder**.

When the browser asks you to choose a folder, choose the matching asset folder for that area.

After saving, refresh the page if you need the new files to appear in the index.

This keeps the information grouped by topic so the full team can see what is there and what still needs done.

## Important

Static local HTML pages cannot silently write files into a folder without browser permission.

That is why the intake pages ask the user to choose a folder when saving. It keeps updates intentional and helps each area stay tidy.

Private integration keys for Make or other services should never be pasted into shared HTML files.
