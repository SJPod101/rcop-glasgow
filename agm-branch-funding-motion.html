Help Branches Grow Again

Vote YES for a sustainable Branch Activity Fund and a stronger democratic mandate for local branch activity.

Position

This motion is nationally supported by branch chairs. It should be presented as positive, professional, member-led and constructive.

What Members Are Being Asked To Support

The resolution number is still to be confirmed when official AGM paperwork is published. The campaign ask is to vote YES for the ordinary resolution establishing a Branch Activity Fund.

2% fund

The Proposal

Ring-fence a minimum of 2% of annual membership subscription income for branch-led activity.

Member value

Why It Matters

Branches deliver local CPD, student engagement, mentoring, outreach, community and professional connection.

Democratic

Oversight

Priorities should be shaped through Branch Chairs and the existing Regional Branch Committee structure.

Key Messages

Keep It Positive

This is about helping branches do more good work locally, not attacking the College.

Membership Control

The fund brings clearer member influence over how a small part of subscription income supports local activity.

National Branch Support

The branch funding motion is not Glasgow-only; it is being driven nationally by branch chairs.

Useful Background

  • Branch levy/funding arrangements changed around 2021; since then branches have lacked consistent refunding certainty.
  • The 2% ask is intended to be proportionate and practical.
  • The message should note inflation carefully if used: historic levy value would be higher today, but the campaign is asking for a modest 2% minimum.
  • Keep emphasis on rebuilding branch activity and supporting members locally.
