Action Requests

A simple visible board for member requests, updates, nudges and decisions. No email pings, no task app, no extra account to manage.

Add a request

Use this when a branch member wants something added, updated, nudged, checked or decided. It goes onto this board so everyone can see it when they have time.

How this board works

Requests are kept here on the visible board. That keeps it light: no automatic email, no task list, no extra system for anyone to manage.

If a request needs a nudge in the group, use Share to WhatsApp. WhatsApp will open with the request text ready, then you choose the Branch WhatsApp group and send it.

Live action board

Requests added on this browser appear here. This is intentionally simple: a visible working list without automatic pings or background emails.
