## Section 1: Motion One

### Help Branches Grow Again

**Vote YES for the Branch Activity Fund Resolution.**

Local branches are where many members experience the practical value of professional community: CPD, mentoring, networking, student support, outreach, and peer connection.

The Branch Activity Fund Resolution would create a sustainable funding framework for local branch activity by ring-fencing a minimum of **2% of annual membership subscription income** for branch-led initiatives.

This is a practical, democratic, member-led proposal designed to strengthen local professional life.

### What the fund could support

- Subsidised local CPD
- Student engagement
- Mentorship programmes
- Public awareness campaigns
- Local networking events
- Outreach and recruitment activity
- Educational workshops

### How priorities would be set

Strategic priorities for the fund would be determined democratically by Branch Chairs through the existing Regional Branch Committee structure.

This supports local accountability, transparent governance, and regional flexibility.

### Why this matters

Since 2021, many branches have operated without standardised funding certainty. This has made it harder to plan local events, deliver education, support volunteers, and build long-term branch activity.

The Branch Activity Fund would help branches plan with confidence and reinvest member subscriptions into visible local value.

### Call to Action

Vote YES to Help Branches Grow Again.

Register for updates so you are ready when AGM voting and proxy guidance is confirmed.

## Section 2: Motion Two

### Member Consultation on The Podiatrist

**Vote YES for proper member consultation.**

This motion concerns the decision to move The Podiatrist to a virtual-only format without first consulting members.

The issue is not whether digital communication has value. Digital access can be useful, efficient, and necessary.

The concern is that a significant change to how members receive a core College publication should involve proper member consultation.

### Suggested Positioning

This motion is about member voice.

The profession can modernise communications while still asking members how major changes should be handled.

### Call to Action

Vote YES to support proper member consultation.

Further details will be added once the full motion briefing and AGM documentation are available.
